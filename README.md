# SPACE-DEBRIS
"**SPACE-DEBRIS**" is a cooperative game set in the "_Quota_" genre, in which players collect the remains of space stations in zero gravity.

# PLAYER CONTROLS
- **LKM** — _grab with your LEFT hand_.
- **RKM** — _grab with your RIGHT hand_.
